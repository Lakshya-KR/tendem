# Hermes Agent framework from NousResearch (github-only; no PyPI wheel).
# PREREQUISITE: `git` must be available on the system PATH for pip to
# resolve this URL. README section 1 documents the prereq.
#
# app/core/hermes_agent.py performs a runtime try/except import. When the
# package installs cleanly, its AIAgent class is bound and every downstream
# agent (Polymarket, Kalshi, Data, Kronos, Kelly, Feedback) uses the real
# framework. If the import raises (e.g. the known `agent.transports`
# runtime error), a Hermes-compatible shim with the same AIAgent interface
# (chat/reset/quiet_mode/ephemeral_system_prompt) is bound as a fallback
# so the pipeline remains operational.
