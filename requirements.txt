# ─── NOTE on the Hermes Agent framework ──────────────────────────────────────
# The Hermes Agent framework (nousresearch/hermes-agent) is intentionally NOT
# listed in this file. It is github-only (no PyPI wheel), so declaring it as
# a dependency here would cause `pip install -r requirements.txt` to fail
# with an opaque error on machines without git. Keeping requirements.txt
# fully pip-compatible is a hard requirement for fresh-environment installs.
#
# To install the real Hermes Agent framework (git MUST be on PATH):
#     pip install -r requirements-hermes.txt
#
# app/core/hermes_agent.py auto-detects the real framework at runtime via a
# try/except import. If the package is importable, its AIAgent class is
# bound at module level and every downstream agent uses it directly. If not,
# a Hermes-compatible shim with the same AIAgent interface is used so the
# pipeline remains operational - a single INFO log line explains which
# path is active.
