================================================================================
  REQUIRED ENVIRONMENT KEYS (minimum to run the full pipeline)
================================================================================

  Copy .env.example to .env and fill in at least these three:

    1. OPENROUTER_API_KEY   - your OpenRouter API key (free tier works)
                              get one at https://openrouter.ai/keys

    2. APIFY_TOKEN          - your personal Apify token (free tier works)
                              get one at https://console.apify.com/settings/integrations

    3. OPENAI_API_KEY       - can be the same value as OPENROUTER_API_KEY;
                              used by the Hermes agent framework

  Everything else (base URLs, model names, paths, Kalshi/Polymarket endpoints,
  history file path, log level, Kronos toggles) has a working default. Only
  override them if you need a proxy, a different model, or a custom path.

  See README.md Section 3 for the full table with defaults and descriptions.

================================================================================
  HOW TO RUN (TL;DR - full instructions in README.md)
================================================================================

   1. git clone https://github.com/Lakshya-KR/tendem.git && cd tendem
   2. python -m venv venv && venv\Scripts\activate   (Windows)
                                                      or: source venv/bin/activate
   3. pip install -r requirements.txt
   4. cp .env.example .env       and fill in the three REQUIRED keys above
   5. python scripts/verify_env.py      (optional - masked audit)
   6. python main.py --asset BTC        (one-shot run)
                  or
      python -m uvicorn app.api:app --reload     (start the API server)

  For Kronos real inference (optional but recommended):
     scripts\setup_kronos.bat   (Windows)   or   bash scripts/setup_kronos.sh

  Again: READ README.md FOR THE FULL ENV KEY TABLE AND TROUBLESHOOTING NOTES.
